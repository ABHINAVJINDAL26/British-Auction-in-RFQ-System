# 🐛 Fix for "Insufficient Permissions"

If you see this error while creating an RFQ:

1. **Stale Session**: You are likely logged in as a **SUPPLIER** from an old test.
2. **Action**: Click the **Logout** button.
3. **Action**: Create a **NEW** account on the **Signup Page**.
4. **Action**: Make sure you click the **"BUYER"** button (It must turn **blue**).
5. **Verify**: Check the header on your Dashboard. It should say:
   - `Buyer Control Center`
   - `BUYER` (in a small green box)

If it still says `SUPPLIER` or gives the error, please send another screenshot!
